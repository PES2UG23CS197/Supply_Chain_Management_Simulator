## Project Description
A simulation system for tracking products through a supply chain from manufacturer to end consumer, including inventory management, logistics optimization, and demand forecasting. Students can implement complex algorithms, data structures, and business logic modeling.

## Instructions

### Prerequisites
- Python
- Streamlit
- MySQL
- Pytest
- Pylint
- Bandit

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/PES2UG23CS197/Supply_Chain_Management_Simulator.git
   cd Supply_Chain_Management_Simulator
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application
   ```bash
   cd src
   streamlit run main.py
   ```
## Testing

```bash
# Run tests
npm test

# Run tests with coverage
npm run test:coverage
```
